# 📄 AI Resume Analyzer

An intelligent web application that analyzes resumes against job descriptions to provide a match score and actionable feedback.

## ✨ Features
- **PDF Parsing:** Automatically extracts text from uploaded PDF resumes.
- **Semantic Matching:** Uses `sentence-transformers` to understand the context of your skills, not just keywords.
- **Instant Feedback:** Provides a compatibility percentage and identifies skill gaps.

## 🛠️ Tech Stack
- **Backend:** Flask (Python)
- **AI/ML:** PyTorch, Sentence-Transformers, Scikit-Learn
- **Deployment:** Render

## 🚀 How to Use
1. Visit the [Live Site](https://ai-resume-analyzer-tfpq.onrender.com).
2. Upload your Resume (PDF format).
3. Get your score and optimize!
